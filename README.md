# Portfolio

A personal portfolio website built using **React** and **Vite**. This website showcases my profile, education, certificates, technical skills, projects, and contact information in a clean and modern layout.

## Live Demo

View the deployed website here:

[Portfolio Website](https://portfolio-seven-drab-14.vercel.app/)

## About the Project

This portfolio website is designed to present my work, skills, and learning journey as a Computer Science student. It includes multiple sections such as About Me, Education, Certificates, Skills, Projects, and Contact. The project is developed using React components and styled with CSS to create a responsive and user-friendly interface.

## Features

* Personal introduction section
* About Me section with profile information
* Education section
* Certificates section with certificate preview modal
* Tech stack section
* Skills section with progress bars
* Projects section with project cards
* Contact section with social media links
* Footer section
* Deployed using Vercel

## Tech Stack

* React JS
* Vite
* JavaScript
* HTML
* CSS
* React Icons
* Font Awesome
* Vercel

## Project Sections

### Home

Introduces me as a Computer Science student and highlights my interest in web development, Python, IoT, and real-world project work.

### About

Provides a short description about my background, skills, learning goals, and the types of projects I have worked on.

### Education and Certificates

Displays my education history, completed certificates, and technology stack.

### Skills

Shows my technical skills in programming languages, frameworks, data science, AI, and tools.

### Projects

Includes project cards for:

* Hospital Management System
* Static Website
* Work Monitoring System
* Smart Air Quality Monitoring System

### Contact

Provides contact details and social media links for communication and collaboration.

## Deployment

This project is deployed using **Vercel**. The production version can be accessed through the live demo link above.
